# 🖥️ Operating System

An **Operating System (OS)** is software that manages and handles hardware and software resources of a computing device.

### 🔹 Key Functions

* Manages computer resources such as CPU, memory, and files
* Acts as an interface between user and hardware
* Performs core functions like process, memory, and file management
* Organizes system resources similar to different departments in a government
* Examples: Linux, Unix, Windows 11, MS-DOS, Android, macOS, iOS

## 📘 Introduction

```
An Operating System acts as an intermediary between the computer hardware and the user.
It prevents the user from directly accessing the hardware and manages all system resources.
```

### ✔️ Features

* Provides an environment to execute programs efficiently
* Runs continuously in the background
* Assigns resources like memory, processors, and I/O devices

---

## 👨‍💻 Operating System and User Interaction

```
Every general-purpose computer consists of hardware, an operating system,
system programs, and application programs.
```

### 🔹 Components

* **Hardware:** CPU, ALU, memory, I/O devices, storage
* **System Programs:** Compilers, loaders, editors, OS
* **Application Programs:** User-level software

### 🔹 Interaction Interfaces

* **CLI (Command-Line Interface):** Bash, PowerShell
* **GUI (Graphical User Interface):** Windows Desktop, macOS Finder

---

## 🎯 Goals of Operating System

### 🔹 Primary Goals

1. **User Convenience**

   * Easy-to-use interface

2. **Program Execution**

   * Provides environment for running programs

3. **Resource Management**

   * Manages CPU, memory, storage, I/O devices

4. **Security**

   * Protects system and user data

---

### 🔹 Secondary Goals

* **Efficient Resource Utilization**
* **Reliability & Stability**

  * Handles errors and exceptions gracefully
  * Modular and easy to debug

## 📜 History of UNIX

* Started in 1964 at Bell Labs
* Successfully developed in 1969
* Created by Dennis Ritchie and Ken Thompson
* First called **UNICS**, later renamed **UNIX**

### 🔹 UNIX Versions & Variants

* IBM → AIX
* Sun → Solaris
* HP → HP-UX
* Apple → macOS

---

## 🤔 Why Linux Was Created?

* Created by **Linus Torvalds** (1991)
* Needed a free alternative to expensive UNIX systems
* Inspired more by **MINIX** than UNIX

### 🔹 About MINIX

* Created by Andrew S. Tanenbaum (1987)
* Designed for teaching OS concepts
* Microkernel-based system

## 🔹 Layers of Linux Architecture

### 1. Kernel

```
Core of the OS that manages hardware and system operations.
```

* Process management
* Memory management
* Device control

### Types of Kernels

* **Monolithic Kernel**
  All operating system services run in a single large kernel space, offering high performance but less modularity.

* **Microkernel**
  Only essential functions run in the kernel, while other services run in user space, improving stability and security.

* **Hybrid Kernel**
  Combines features of monolithic and microkernels to balance performance and modularity.

* **Exokernel**
  Provides minimal abstraction, giving applications direct access to hardware for maximum flexibility and performance.

---


### 2. System Libraries

```
Provide functions for applications to interact with the kernel.
```

* Reusable code
* Interface between apps and kernel

---

### 3. Shell

```
Command-line interface for user interaction.
```

* Executes commands
* Connects user and kernel

---

### 4. Hardware Layer

```
Physical components of the system.
```

* CPU, RAM, storage
* I/O devices

---

### 5. System Utilities

```
Tools for system management and maintenance.
```

* Software installation
* User management
* System monitoring
